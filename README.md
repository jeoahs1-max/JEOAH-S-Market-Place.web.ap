# JEOAH'S - Place de Marché

## Présentation

JEOAH'S est une place de marché conçue pour connecter les vendeurs, les affiliés et les acheteurs. Notre objectif est de fournir les outils et la visibilité nécessaires pour que chacun puisse prospérer.

## Offres et Rôles

### Offre de Lancement
Pour célébrer notre lancement, les **25 premiers utilisateurs inscrits** (tous rôles confondus) bénéficieront d'un **essai gratuit de 15 jours** sur le plan d'abonnement mensuel.

### Rôles des Utilisateurs
1.  **Acheteur** : Parcourt la marketplace et achète des produits. Des frais de service de **3%** s'appliquent sur chaque achat pour sécuriser la transaction.
2.  **Vendeur** : Met en vente ses produits. Peut choisir un plan d'abonnement pour l'automatisation par IA ou utiliser le plan gratuit en gérant sa propre promotion.
3.  **Affilié** : Promeut les produits via des liens uniques pour gagner des commissions.

### Programme de Parrainage
Chaque utilisateur dispose d'un lien de parrainage. Partagez-le pour recevoir des commissions sur les abonnements souscrits par vos filleuls. Ces commissions sont cumulables et retirables.

## Informations Clés
-   **Retraits** : Un solde minimum de 100 USD est requis pour le premier retrait. Un montant équivalent à votre abonnement mensuel est conservé comme solde de réserve.
-   **Frais** : JEOAH'S n'applique aucun frais sur les retraits ; seuls les frais du fournisseur de paiement externe (PayPal, Wise, etc.) sont déduits.
