Politique de Confidentialité

Dernière mise à jour : 28 Octobre 2025

1. Collecte et Utilisation des Informations

JEOAH'S s'engage à protéger la confidentialité de vos informations. Nous collectons votre nom, e-mail, rôle (Acheteur, Vendeur, Affilié) et les données de transaction pour vous fournir les services de la Marketplace. Vos données de rôle et de transactions sont stockées dans Firestore sous votre identifiant utilisateur unique (`/artifacts/{appId}/users/{userId}/...`).

2. Sécurité des Données (Firebase)

Nous utilisons l'infrastructure Firebase pour l'authentification et le stockage de vos données. L'accès à vos données est sécurisé par les règles de sécurité Firebase, garantissant que seul vous (et les administrateurs avec un accès strict) pouvez consulter vos informations privées.

3. Code de Parrainage (JSMP-)

Votre code de parrainage JSMP- est unique et public. Il est utilisé pour le suivi des affiliations. L'utilisation de ce code ne révèle aucune information personnelle sur vous, uniquement qu'un affilié vous a référé.
